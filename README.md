# 👨‍💻 André Caldas

**`Reverse Engineering | Cybersecurity | Low-Level `**

Aqui você encontrará RE **baixo nível**, explorando como softwares realmente funcionam “por baixo dos panos”: desde o **loader**, **runtime**, **memória**, **assembly**, até **mecanismos de proteção, anti-debug e evasão**. 

---

## 🧠 Áreas de Foco

- 🔍 Engenharia Reversa de Executáveis (Windows / Linux)
- 🧬 Análise Estática e Dinâmica
- 🧪 Malware Analysis (nível iniciante → avançado)
- 🧠 Assembly (x86 / x64)
- 🛡️ Anti-Debugging & Anti-Analysis
- 🌐 Redes, protocolos e comportamento de tráfego
- ⚙️ Sistemas Operacionais e gerenciamento de memória
